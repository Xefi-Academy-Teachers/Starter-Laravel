### Usefull commands

Start containers
```shell
./vendor/bin/sail up
```

Enter containers
```shell
./vendor/bin/sail shell
```

Recreate database quickly with seeding
```shell
./vendor/bin/sail php artisan migrate:fresh --seed
```

### Required commands

Always having one terminal running
```shell
./vendor/bin/sail npm run dev
```
